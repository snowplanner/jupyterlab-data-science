The module 'fundamental_functions' contains several function modules performing more advanced calculations and a notebook giving an example on how to employ the previously stated functions

In this folder, you will encounter the following modules/notebooks:
  1. 'dem_utils_import.py': 
    - import functions from module 'dem_utils'
  
  2. 'fundamental_functions_import.py':
    -import functions from module 'fundamental_functions'

  3. 'crop_raster.py': crop a raster employing a shapefile
    - 'flatten_raster_two_dim'
    - 'mask_by_shape'
  
  4. 'classify_rasters.py': creates a classified raster from input conditions
    - 'gdal_processing'
    - 'classify_raster'
  
  5. 'points_into_polygons.py': polygonizes a raster and classifys whichp points lie inside a polygon
    - 'polygonize_raster'
    - 'points_into_polygons'
    - 'pretty_gdf'
  
  6. 'statistics_kram.py': calculates the statistics of a geodataframe
    - 'stat_analysis_sd'

  7. 'test_module.ipynb': tests the previous functions
