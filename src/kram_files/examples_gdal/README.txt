The module 'examples_gdal' contains several notebooks giving some examples on how to use the basic functionalities implemented on the folder dem_utils 

In this folders, you will encounter the following notebooks:
  1. 'gdal_operations.ipynb': 
    - from an initial dem image, different calculations are performed
    - manual reclassifications of a raster

  2. 'raster_shp_basics.ipynb': 
    - crop a dem raster with a shapefile
    - testing of the deletion of a third coordinate

  3. 'mix_rasters.ipynb': 
    - creation of a classified raster employing the slope, aspect and height

  4. 'sort_points2polygons.ipynb': 
    - polygonization of a raster
    - spatial join between points and polygons
